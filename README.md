# Projet de synthèse – M2 CHPS

## Objectif
Ce projet vise à étudier la propagation d'une maladie à l’aide :
- d’un modèle épidémiologique SEIRS (ODE)
- d’un modèle multi-agent (SMA)

## Contenu
- Partie 1 : Résolution numérique du modèle SEIRS
- Partie 2 : Modèle multi-agent en C, C++ et Python
- Analyse et visualisation des résultats avec Python (Jupyter)
